Gather Words Using the SIL Comparative African Wordlist

This tool is a collection of 1700 English words gathered by SIL that you can scan through. When you find a word you know in the language you are working on, you can add it.


Use Page Up and Page Down or the two arrows to scroll through the words; type the word and press Enter to add a word. It will appear in the empty box. You can add multiple entries for each word in the database. Later on you can go through and put in more information for these words separately.
