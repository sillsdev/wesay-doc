Export to SFM

This is a way of saving your dictionary in a standard form that was previously used in Toolbox (for more info, see http://www.wesay.org/wiki/Notes_on_Export_To_SFM).  If you click Set Up you can change the languages that this export will use, as well as enter exceptions and rules. You can also toggle the visibility of this button in WeSay. When the button is in WeSay, if you click it a text document will pop up containing your entire dictionary in SFM format, with the appropriate tags in place.
