Actions

There are five possible actions to select in the WeSay Configuration Tool, shown below. All of these actions require some setup and are for use in storing, sharing or processing your dictionary, rather than building more entries. The Configuration Tool features some explanations of these functions, but you can read explanations here as well.


After you have set up the tools, you can use their functions either by opening WeSay and using the buttons there, or by clicking the icons in the WeSay Configuration Tool.
