Gloss vs. Definition

In WeSay we prefer to  use definition over gloss, which is intended for use with inter-linear texts.  You have the option to enable glosses, but by default the program is set up to only use definitions.


The word descriptions entered in definitions can be many words, much longer than a typical gloss, but they don't necessarily have to be. For a dictionary that is translating from one language to another, one or two word definitions are often sufficient.
