Dictionary Browse & Edit

This tool is used to search through all of your dictionary entries to find certain words and edit any incompletions.


There are the basic options New Word and Delete This Word for adding to or deleting a word from your dictionary. The Show Uncommon Fields button reveals several otherwise hidden fields, which don't normally appear so that the page doesn't get cluttered. You can change what fields are hidden in the Fields menu in the Configuration Tool.

When using the search function, your input does not have to exactly match the word you are searching for. You only need to type in a word similar to the one you are looking for, and the search function will automatically bring up a list of close approximations, as shown below.
