Make PDF Dictionary

This option converts your dictionary to a PDF, which makes it easier to print. To do this you will need the PrinceXml engine (for more information visit www.princexml.com or www.wesay.org/wiki/MakePdfDictionary). You can use PrinceXml free unless you are intending to use your dictionary for commercial use, in which case you will need to purchase a license. Note that this feature is not for final publication, but is intended as an easy way to print out your dictionary as you are building it.
