Getting the APT Lines

There are three APT lines you will need to add to the Synaptic Package Manager in Ubuntu in order to have access to WeSay. The lines are different for each version of Ubuntu – Hardy, Intrepid and Jaunty. You can find the line specific to your version of Ubuntu at http://projects.palaso.org/wiki/wesay.

Or you can use the APT lines here, which are for Jaunty:

deb http://packages.sil.org/ubuntu jaunty main

deb http://packages.palaso.org/ubuntu jaunty main

deb-src http://packages.palaso.org/ubuntu jaunty main
